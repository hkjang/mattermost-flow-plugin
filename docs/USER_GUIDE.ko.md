# 사용자 가이드

이 문서는 Mattermost 안에서 Flow 보드, 대시보드 뷰, 간트 뷰를 사용해 업무를 관리하는 사용자를 위한 가이드입니다.

## Flow 열기

다음 방법으로 플러그인을 열 수 있습니다.

- 채널 헤더의 `Open Flow board` 버튼 클릭
- `/flow open` 실행
- 보드, 간트 뷰, 카드 딥링크 열기

채널 안에서 Flow를 열면 현재 팀과 채널 문맥이 자동으로 적용됩니다.

## 보드 생성과 선택

Flow는 팀 범위 보드와 채널 범위 보드를 모두 지원합니다.

- 팀 보드: 같은 팀 전체가 함께 사용하는 보드
- 채널 보드: 특정 채널에 연결되어 프로젝트나 운영 흐름에 적합한 보드

채널에 기본 보드가 연결되어 있으면 `/flow open`과 빠른 카드 생성은 그 보드를 우선 사용합니다.

## 보드 뷰

보드 뷰는 칸반 방식으로 일감을 다루는 메인 화면입니다.

주요 작업:

- 기본 컬럼으로 새 보드 생성
- 원하는 컬럼에 카드 추가
- 컬럼 사이로 카드 드래그 이동
- 같은 컬럼 안에서 카드 순서 조정
- 담당자, 라벨, 상태, 날짜 범위 필터
- 공유 가능한 보드 링크 복사

카드에는 다음 정보를 담을 수 있습니다.

- 제목과 설명
- 보드 범위 안의 Mattermost 사용자 담당자
- 라벨과 우선순위
- 시작일과 마감일
- 진행률과 마일스톤 여부
- 체크리스트
- 첨부 링크
- 댓글

## 카드 템플릿

보드 관리자는 보드 설정에서 재사용 가능한 카드 템플릿을 저장할 수 있습니다.

템플릿으로 미리 채울 수 있는 항목:

- 카드 제목과 설명
- 라벨과 우선순위
- 상대 시작일과 마감일
- 마일스톤 여부
- 체크리스트
- 첨부 링크

빠른 카드 생성 바에서는 먼저 템플릿을 고른 뒤 제목, 마감일, 담당자, 우선순위를 추가로 조정해서 카드를 만들 수 있습니다.

## 간트 뷰

간트 뷰는 보드 뷰와 같은 카드와 날짜 데이터를 사용합니다.

할 수 있는 작업:

- 보드 뷰와 간트 뷰를 오가며 일정 확인
- 타임라인 위에서 일정 시각화
- 작업 막대 가운데를 드래그해서 일정 이동
- `Start`, `End` 핸들을 드래그해서 기간 조정
- 간트 행에서 카드 바로 열기
- 공유 가능한 간트 링크 복사

## 대시보드 뷰

대시보드 뷰는 현재 보드의 운영 상태를 빠르게 파악할 수 있는 요약 화면입니다.

확인할 수 있는 항목:

- 전체 카드, 완료 카드, 지연 카드, 마감 임박 카드, 마일스톤, 평균 진행률 같은 핵심 지표
- 컬럼별 상태 분포
- 우선순위 분포
- 진행 중 업무 기준 담당자 load
- 곧 마감되는 카드
- 마일스톤 카드
- 최근 보드 활동

대시보드는 현재 필터를 그대로 반영하므로 특정 담당자, 라벨, 상태, 기간 기준으로 좁혀서 볼 수 있습니다.

## 캘린더 연동

Flow는 일정이 있는 카드를 iCalendar 피드로 내보낼 수 있습니다.

할 수 있는 작업:

- 상단 툴바에서 현재 보드를 `.ics` 캘린더로 열기
- 보드 설정에서 외부 캘린더 구독 링크 활성화
- Google Calendar, Apple Calendar, Outlook에 구독 URL 붙여넣기
- 기존 공유 링크를 폐기해야 할 때 구독 토큰 재발급

공개 구독 URL은 보드 관리자가 보드 설정에서 캘린더 피드를 활성화했을 때만 사용할 수 있습니다.

## 카드 상세

카드를 선택하면 상세 패널이 열립니다.

카드 상세에서 가능한 작업:

- 메타데이터와 날짜 수정
- 담당자 추가 또는 제거
- 체크리스트 편집
- 댓글 추가
- 카드 간 의존성 링크 추가
- 활동 이력 확인
- 카드 딥링크 복사

## Mattermost 안에서의 협업

보드 설정에서 허용하면 Flow가 연결된 채널에 업데이트를 직접 포스트합니다.

예시:

- 카드 생성
- 카드 이동
- 마감 임박 알림

Flow 포스트에는 다음과 같은 빠른 액션이 표시될 수 있습니다.

- `Assign to me`
- `Move to next`
- `Push +1 day`
- `Push +1 week`
- `Complete next item`
- `Mark done`
- `Open card`
- `Open gantt`

이 액션은 보드와 열려 있는 Flow 화면에 실시간으로 반영됩니다.

## 슬래시 명령

빠른 접근을 위해 기본 슬래시 명령을 사용할 수 있습니다.

```text
/flow open
/flow new Ship release --due 2026-04-10
/flow help
```

`/flow new`는 현재 보드 범위에 카드를 생성합니다. 채널에 기본 보드가 연결되어 있으면 그 보드를 먼저 사용합니다.

## 팁

- 특정 채널 대화 흐름과 강하게 연결된 일감이면 채널 보드를 사용하세요.
- 여러 채널이 같은 계획 화면을 봐야 하면 팀 보드를 사용하세요.
- 다른 사람을 정확한 문맥으로 이동시키고 싶다면 보드 링크와 카드 링크를 공유하세요.
- 외부 이해관계자에게 읽기 전용 일정이 필요하면 캘린더 구독 URL을 활용하세요.
- 카드에 마감일을 넣어 두면 간트 뷰와 마감 임박 알림이 훨씬 유용해집니다.

## 문제 해결

- `/flow open` 실행 시 보드가 없다고 나오면 해당 범위에 먼저 보드를 만드세요.
- 담당자 검색 결과가 보이지 않으면 사용자가 해당 팀 또는 채널 멤버인지 확인하세요.
- 포스트 액션으로 카드를 바꿨는데 화면이 늦게 보이면 보드 링크를 한 번 다시 열어 보세요. 보통 자동 동기화되지만 새로 열기는 빠른 fallback입니다.
